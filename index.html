Warum ist Windows besser als Linux?

-Windows ist aelter -Windows hat Start-Menu -Windows hat sehr leichten und sehr beautiful UI -Windows hat besseren Logo -Windows ist mit fast allem kompatibel -Windows hat besseren geschichte -Windows hat sehr guten und ausgebildeter File Explorer (der von Linux ist extrem scheisse) -Windows hat besseren Boot und besseren Log on -Windows ist so customisable Vertraue Windows, der beste OS der Welt.
